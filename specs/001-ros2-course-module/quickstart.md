# Quickstart Guide: Module 1 – The Robotic Nervous System (ROS 2)

## Overview
This quickstart guide will help you understand and use Module 1 of the Physical AI & Humanoid Robotics course, which introduces ROS 2 as the core middleware for humanoid robotic systems.

## Prerequisites
- Basic knowledge of AI and Python programming
- Familiarity with fundamental computer science concepts
- Access to the course materials hosted on the Docusaurus documentation site

## Learning Objectives
After completing this module, you will be able to:
1. Explain the role of ROS 2 as middleware in humanoid robotics
2. Describe how nodes communicate using topics and services
3. Understand how Python-based AI logic interfaces with ROS controllers
4. Conceptually understand humanoid URDF files

## Module Structure
The module consists of four interconnected chapters:

### Chapter 1: Introduction to ROS 2 and Middleware for Robots
- Understand ROS 2 as the "nervous system" of robots
- Learn about the distributed architecture of ROS 2
- Explore the advantages of using middleware for robotics

### Chapter 2: ROS 2 Core Concepts: Nodes, Topics, and Services
- Define and understand ROS 2 nodes
- Explain the publish-subscribe pattern with topics
- Understand request-response communication with services

### Chapter 3: Bridging Python AI Agents to Robots using rclpy
- Connect Python-based AI logic to ROS 2 systems
- Learn how to create Python nodes using rclpy
- Understand the interface between AI algorithms and robot controllers

### Chapter 4: Understanding Humanoid Structure with URDF
- Learn about Unified Robot Description Format (URDF)
- Understand how robots are represented in XML format
- Explore the components of humanoid robot structure
